## 🎨 Guía de Estilo para el Proyecto

### Tecnologías de Estilo
- Tailwind CSS para estilos utilitarios
- Componentes con clases de Tailwind
- Sistema de diseño consistente

### Paleta de Colores
- **Color primario**: #3b82f6 (azul)
- **Color secundario**: #10b981 (verde)
- **Fondo principal**: #f9fafb (gris muy claro)
- **Fondo de tarjetas**: #ffffff (blanco)
- **Texto principal**: #111827 (casi negro)
- **Texto secundario**: #6b7280 (gris medio)
- **Bordes**: #e5e7eb (gris claro)

### Tipografía
- **Títulos**: Inter, sistema sans-serif
- **Cuerpo**: Inter, sistema sans-serif
- **Datos numéricos**: JetBrains Mono, monospace

### Espaciado
- Usar escala de 4px (Tailwind: 1 = 4px)
- Padding estándar en tarjetas: p-4 o p-6
- Gap entre elementos: gap-4

### Estilos de Componentes
- **Tarjetas**: fondo blanco, border-radius: 0.5rem (rounded-lg), sombra suave (shadow-sm)
- **Botones**: padding: 0.5rem 1rem (px-4 py-2), border-radius: 0.375rem (rounded-md)
- **Inputs**: borde sutil (border border-gray-300), focus: ring-2 ring-blue-500

### Modo Oscuro (Opcional)
- Implementar con clase `dark:` de Tailwind
- Fondo oscuro: #1a1a1a
- Tarjetas oscuras: #2d2d2d
